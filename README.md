=======================系统概要=======================

当前系统波特率采用的是115200，系统时钟50Mhz

AD采样率为1.5Mhz，AD时钟信号采用PLL分频45Mhz，因为AD7352时钟最大允许上限为48Mhz

FIFO空间为12*16384，写时钟45Mhz，读时钟50Mhz
